O Problema

A rotina era travada por um trabalho manual que tomava muito tempo do operador. A necessidade de baixar arquivos, analisar localmente e enviar por email manualmente, criava um fluxo ineficiente e propenso a baixa assertividade.

> STATUS: CRITICAL_BOTTLENECK
JS

Fullstack JavaScript: Do Frontend ao controle do Sistema Operacional.

LOCAL FIRST

Manipulação direta de arquivos (File System) sem latência de rede.

A Solução (V1.1.6)

O SIAJ Desktop nasceu para rodar nativamente. Utilizando o poder do Node.js no backend local, o sistema acessa o site do TJ-Go, baixa PDFs pesados instantaneamente, faz a análise, resumo e envio automático.

// CORE FLUX: DO CAOS À ORDEM

1. A Fonte: DJE

O Diário da Justiça Eletrônico é um "Data Lake" não estruturado. Arquivos PDF com milhares de páginas, contendo intimações, sentenças e despachos misturados.

> INPUT: .PDF (VECTOR/TEXT)
> VOLUME: ~60.000 PÁGINAS/DIA
REGEX

MINERADOR

O sistema varre o arquivo bruto utilizando Expressões Regulares (Regex) complexas para identificar padrões:

  • [x] Nomes de Advogados
  • [x] CPF de Pessoas
  • [x] Números de Processo
  • [x] Termos Jurídicos Chave

2. Triagem Lógica

Não basta achar o nome. O SIAJ analisa o contexto.

Ele diferencia uma "Intimação de Audiência" de uma simples "Suscitação de Dúvida", categorizando a urgência automaticamente.

3. Entrega (Delivery)

O resultado final não é apenas um dado, é uma Ação.

Instantâneo

Alerta imediato no WhatsApp configurável, com o bloco de páginas relevantes e resumo do ato.

Formal

Email com o PDF recortado (apenas as páginas relevantes) em anexo junto ao resumo do ato.

// SECURITY CORE: DRM & LICENSING

PROTEÇÃO DE PROPRIEDADE INTELECTUAL

Hardware Binding (HWID)

O sistema utiliza uma arquitetura de Fingerprint Digital. Ao ser instalado, o software escaneia a assinatura física da máquina (Placa-mãe, CPU e UUID do disco).

Isso gera um Hash Único. A licença só funciona naquela máquina específica. Se o usuário copiar a pasta para outro computador, o hash não bate e o sistema entra em modo de bloqueio imediato (Lockdown).

AUTO UPDATER

Integração com GitHub Releases. O software detecta novas versões na nuvem, baixa o patch e atualiza silenciosamente (OTA) na máquina do cliente.

Engenharia Anti-Bloqueio

Para garantir o envio de mensagens em massa sem instabilidade, implementamos um Chromium Engine Isolado via ASAR Unpack.

Utilizamos técnicas de DOM Injection para manipular a interface do WhatsApp Web diretamente, contornando limitações de API e falhas de renderização.
